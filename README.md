## Описание
Набор unit-тестов для проверки корректности работы функций вычисления площади и периметра окружности. Тесты позволяют автоматически убедиться в правильности математических вычислений при различных входных данных.

## Возможности
- Проверка функции вычисления площади круга (`area`)
- Проверка функции вычисления длины окружности (`perimeter`)
- Тестирование нулевых и положительных значений радиуса
- Использование `assertEqual` и `assertAlmostEqual` для точного сравнения результатов
- Автоматическое выполнение всех тестов с помощью `unittest`
- Понятный отчёт о прохождении или провале тестов
